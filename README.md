# My Qtile dotfiles for Arch linux


## My soft
- **Shell**: fish
- **Web brower**: chromium
- **File manager**: thunar
- **Fetch**: neofetch, [moonfetch](https://github.com/d4yr41n/moonfetch)
- **Messenger**: kotatogram
- **Pdf-reader**: evince
- **Differnt**: notion

## Yay install

```
git clone https://aur.archlinux.org/yay.git  
cd yay  
makepkg -si
```
